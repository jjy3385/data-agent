# MVP 문서 안내

> 이 폴더는 현재 3주 MVP에서 어디까지 구현하고 무엇을 다음 단계로 미루는지 관리한다. 방향이나 우선순위가 헷갈릴 때 `scope.md`와 `roadmap.md`를 먼저 확인한다.

## 문서 목록

| 문서 | 설명 |
|---|---|
| [MVP 범위](scope.md) | 대표 시나리오, 역할, Metadata와 범용성 경계 |
| [시나리오 후보](scenario-candidates.md) | AdventureWorks2022 데이터 기반 대표 질문 선정 근거 |
| [MVP 로드맵](roadmap.md) | 주차별 구현 순서와 현재 작업 목록 |
| [MVP 완료 기준](acceptance-criteria.md) | Golden Query와 E2E 완료 판정 조건 |

## 사용 원칙

* 기능을 추가하기 전에 `scope.md`의 범위 안인지 확인한다.
* 구현 순서를 바꾸면 `roadmap.md`를 먼저 갱신한다.
* “동작한다”가 아니라 `acceptance-criteria.md`의 조건을 만족해야 완료로 판단한다.
